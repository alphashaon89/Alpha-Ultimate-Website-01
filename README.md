# Alpha Ultimate Ltd. Website

This project is a modern, multi-page website for Alpha Ultimate Ltd., a premium cleaning service company.

## Features

*   **Service Presentation:** Detailed descriptions of available cleaning services.
*   **Price Estimation:** An interactive tool for users to get a price estimate based on their needs.
*   **Booking:** A streamlined form for booking services.
*   **Visual Gallery:** A showcase of before-and-after images and videos.
*   **Multi-page Navigation:** A complete website structure with Home, Services, Pricing, Gallery, Booking, and Contact pages.
*   **Modern UI/UX:** Clean design, smooth animations, and a professional look and feel.
